正则化

正则化可视为能有效得降低模型参数在不同训练集采样分布下的方差（同时较小得増加偏差），从而加强了模型的泛化能力。
事实上，实践中的很多问题的真实数据生成过程都非常复杂。以至于不论多复杂的模型族都难以包括真实的数据生成过程。所以说，如果想仅仅通过找到更合适复杂度的模型来提高模型的泛化能力往往效果不佳。相反，在实践中往往发现，一个适当正则化的大型模型是泛化能力最好的模型。
正则化的一般做法是对目标函数J添加一个参数范数惩罚项$\Omega(\theta)$,我们将正则化后的目标函数记为：$$\widetilde{J}(\theta;X,y) = J(\theta;X,y)+\alpha\Omega(\theta)$$选择不同的范数$\Omega$会产生不同的参数偏好，下面我们讨论L2，L1范数对模型参数的偏好。

$L^2$正则化

简单起见，使用一个没有偏置项的线性模型作为例子，使用L2正则化的目标函数可以表示为：$$\widetilde{J}(w;X,y) = \frac{\alpha}{2}w^T w+J(w;X,y)$$与之对应的梯度为$$\nabla_w\widetilde{J}(w;X,y) = \alpha w + \nabla_w J(w;X,y)$$用梯度下降法更新权重有：$$w\gets (1-\epsilon\alpha)w - \epsilon\nabla_w J(w;X,y)$$可以看到在每一步更新中都会先收缩权重向量，再执行根据代价函数的梯度更新。
接下来看看在整个训练过程正则化带来的影响。假设$w^* = arg\;min_w J(w)$。我们在$w^*$的邻域内对代价函数进行二次近似：$$\hat{J}(\theta)=J(w^*) + \frac{1}{2}(w-w^*)^T H(w-w^*)$$因为在$w^*$邻域内梯度近似为0，所以省略了一阶项。我们以此近似函数项对w取梯度有：$$\nabla_w \hat{J}(w)=H(w-w^*)$$如果加上L2正则化项并以$\widetilde{w}$表示目标函数最优点，则有：$$\alpha\widetilde{w}+H(\widetilde{w}-w^*)=0$$ $$\widetilde{w}=(H+\alpha I)^{-1}H w^*$$因为Hessian矩阵是实对称的，所以H必可正交分解，也即有$H = Q\Lambda Q^T$所以上式可化为：$$\begin{split}\widetilde{w}&=(Q\Lambda Q^T + \alpha I)^{-1}Q\Lambda Q^T w^* \\&=[Q(\Lambda + \alpha I)Q^T]^{-1}Q\Lambda Q^T w^* \\ &=Q(\Lambda +\alpha I)^{-1}\Lambda Q^T w^*\end{split}$$可以看到$\widetilde{w}$是由将$w^*$投影到H特征向量作为坐标轴的空间中，对每个轴上的坐标以$\frac{\lambda_i}{\lambda_i + \alpha}$为因子进行缩放，再投影回原空间。所以在H特征值较大的方向（$\lambda_i >> \alpha$）正则化影响较小。而对H特征值较小的方向正则化影响较大。如果$\lambda_i <<\alpha$则正则化会将该分量拉至接近于0。

如图，J在w1方向梯度较小，也即对应的J的Hessian矩阵特征值较小，受正则化影响较大，反之，在w2方向影响较小。（横轴w1,纵轴w2）也就是，正则化会使目标函数没有强烈偏好的权重衰减，而对明显影响目标函数权重几乎不变。因此，正则化降低了模型参数对训练集差异的敏感性，从而预防了过拟合的风险。（但过大的正则化系数可能造成欠拟合风险）

$L^1$正则化

L1正则化的正则化项可以被表示为：$$\Omega(\theta)=||w|| = \sum_i |w_i|$$即参数的绝对值之和。还是使用没有偏置的线性模型作为例子。使用了L1正则化的目标函数可以写为：$$\widetilde{J}(w;X,y) = \alpha ||w||_1 +J(w:X,y)$$我们假设$w^*$是J的最优点（未正则化的目标函数），设H是J在该点的Hessian矩阵。为了简化表示，设H是对角矩阵（事实上如果输入特征之间是独立的话，这个假设是成立的。比如对数据使用了主成分分析处理过了）。因为最优点Hessian矩阵是正定的，所以也有$H_{i,i}$。如我们在上一节推导得那样，我们可以将L1正则化目标函数在$w^*$的邻域内二次近似的表视为：$$\widetilde{J}(w;X,y) = J(w^*;X,y)+\sum_i (\frac{1}{2}H_{i,i}(w_i-w_i^*)^2 + \alpha |w_i|)$$要使右式第二项的值取到最小则对每一维i,$w_i$应取：$$sign(w_i^*)\;max(|w_i^*|-\frac{\alpha}{H_{i,i}},0)$$现在考虑$w_i^* >0$的情况:
(1) 如果$w_i^* \le \frac{\alpha}{H_{i,i}}$。正则化的最优$w_i$是0，因为在这个方向取$w_i^*$,J对目标函数的减小量不如正则化项的增加量。因此L1正则化将$w_i$推向0。
(2) 如果$w_i^* \ge \frac{\alpha}{H_{i,i}}$则$w_i$的最优值不为0，而是取$w_i^*$向0方向移动了$\frac{\alpha}{H_{i,i}}$的值。
可以看到L1正则化倾向于将把对目标函数贡献不明显的参数取为0（$\alpha$越大越明显）,而不仅仅像L2正则化按比例减小相关参数。因此L1正则化更易得到“稀疏解”。从而淘汰到对应参数为0的特征，这种方式也称作“嵌入式特征选择”。

以上是参数范数惩罚正则化常用的两种惩罚项。在神经网络领域中还有很多其他正则化策略，下次再写吧。
